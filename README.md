## 🧠 什么是 OpenClaw？

OpenClaw 是一个 **开源、自托管 AI 助手 / 代理 框架**，支持将大型语言模型（如 Claude、GPT、Gemini、本地模型等）连接到你喜爱的聊天与消息平台，实现自动化任务和智能助手功能，而不仅仅是一个聊天机器人。:contentReference[oaicite:0]{index=0}

核心特点包括：

- 🔓 **开源 & 自托管**：完全掌控你的数据和部署环境，MIT 许可，无锁定。:contentReference[oaicite:1]{index=1}  
- 🔌 **多平台集成**：支持 WhatsApp、Telegram、Discord、Slack、Slack、Teams 等 50+ 渠道。:contentReference[oaicite:2]{index=2}  
- ⚙️ **丰富的技能生态**：成千上万的社区技能（skills）可扩展功能。:contentReference[oaicite:3]{index=3}  
- 🔒 **隐私优先**：对话和数据只存储在你自己的服务器或设备上。:contentReference[oaicite:4]{index=4}  
- 🧠 **模型无锁定**：根据需要配置 OpenAI、Anthropic 或本地模型。:contentReference[oaicite:5]{index=5}
